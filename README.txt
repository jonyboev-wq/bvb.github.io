
BVB — готовая сборка
====================

Файлы:
- index.html — главная, сетка карточек, соцкнопки по центру, переключатель темы.
- product.html — страница товара с галереей, кнопками Avito и «Опт».
- wholesale.html — форма оптовой заявки с отправкой в Telegram.
- style.css — стили (светлая/тёмная тема, ровная сетка).
- shopConfig.sample.json — пример конфига товаров/соцсетей.
- favicon.ico — иконка сайта.

Как развернуть:
1) Скопируйте все файлы в корень репозитория GitHub Pages (bvb.github.io).
2) (Опционально) Создайте `shopConfig.json` по образцу `shopConfig.sample.json`.
   Если `shopConfig.json` отсутствует, сайт покажет встроенный набор товаров (fallback) и ничего не «сломается».
3) Для страницы товара используется параметр `?slug=...` (из поля `slug` в конфиге).
4) Телеграм-оповещение в wholesale.html уже настроено (token/chat_id). Если нужно — поменяйте в коде.

Удачи!
